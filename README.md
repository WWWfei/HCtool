# HC-tool
HC-tool是一个在线编辑的可视化布局系统，旨在通过可视化页面来生成前端布局及样式代码。通过此工具，可以设置页面布局、添加元素、设置元素样式从而生成最终前端代码，具有在线编辑和即时预览等功能。

```
npm install
```
